# Trait Flow Prototype Pitch Deck (English)

> Purpose: 10-minute presentation outline for stakeholders and pilot users.  
> Format: One section per slide, bullet-style notes.

---

## 1. Title Slide
- Title: **“Trait Flow: Daily Coaching for Sustainable Self-Awareness”**
- Subtitle: Introducing the small-start MVP prototype
- Presenter name / date

---

## 2. Why Now? (Problem & Context)
- Rising need for mental health and self-management support.
- Existing apps often feel generic or require heavy commitment.
- AI can help, but it needs personal context that updates daily.

---

## 3. Solution Overview
- Trait Flow combines a quick personality snapshot with daily mood check-ins.
- Daily ritual (about 3 minutes):
  1. Ten-item personality assessment (TIPI)
  2. Daily mood & energy check
  3. AI-generated, context-aware coaching tip
- Designed to deliver small but meaningful insights every day.

---

## 4. Demo Journey (User Flow Visual)
- Four mock screens: TIPI → Home → Check-in → Intervention card.
- Caption each:
  - TIPI: Understand your personality in minutes.
  - Home: See today’s status at a glance.
  - Check-in: Log mood/energy in under a minute.
  - Intervention: Receive a tailored prompt or action.

---

## 5. Core Value
- **Deeper self-awareness** via personality × mood timeline.
- **Actionable nudges** matched to the user’s current energy.
- **Feedback loop**: users rate each message; the system adapts over time.

---

## 6. Tech Snapshot
- Frontend: Next.js / React / TypeScript
- Backend: Supabase (PostgreSQL + Edge Functions)
- AI: OpenAI Responses API with structured JSON output
- Security: Supabase Auth with Row Level Security

---

## 7. Roadmap (Progress & Plan)
- Weeks 1-2: Onboarding (TIPI) feature complete
- Week 3: Check-in flow + intervention generation live
- Week 4: History, feedback, basic metrics
- Weeks 5-6: Internal testing + iteration
- Weeks 7-8: Pilot launch with 5–10 users

## 8. Future Vision
